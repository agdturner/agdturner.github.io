Introduction to Python Programming for Geography and Social Science

Welcome! These web pages are Open Educational Resources designed to help with learning the basics of computer programming for geography and social science using the Python programming language (Python).

You will learn about general computer programming concepts including:

  • Variables
  • Control flow(conditionals, for loops)
  • Classes, functions/methods
  • You will practice programming (coding) in Python to:
    • Input data into programs from files and from the Web
    • Output data to files
    • Handle coordinates in lists
    • Display spatial data
    • Process raster data
    • Develop and visualise a dynamic spatial simulation
  • Learning to program involves practical problem solving andcomputational thinking.
  • Programming well involves a basic workflow that is repeated
  • lots of times. The steps are: plan, write, test, review.
    • writing both comments in source code that make it easier to understand what parts of code are for, and writing documentation for users and others that might be interested in collaborating in code development.
  • Understand generic programming tasks and how to go about these in Python including:
    • Testing
    • Documenting
    • Sharing
  • Familiarisation with:
    • Python language development
    • Python limitations
    • Jupyter Notebooks
    • Version control
    • GitHub

    • The Learning Journey

    • Take time to develop your understanding and practise your skills.
    • There is a lot to learn especially if you are new to programming.
    • Repeating tasks can help you learn.
    • Try to learn new things when you are fresh and try to take plenty of breaks to rest from computer work and to allow new ideas and concepts to sink in.
    • Talk to others without disturbing them. Work together if you like.
    • Ask your tutor for help or clarification in class or via email.
    • Test your understanding by testing that your code does what you expect it to do.
    • Be experimental.
    • Learn to interpret error messages.
