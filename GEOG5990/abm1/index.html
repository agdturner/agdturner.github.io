Agent Based Model Part 1

1. Introduction

In a series of practicals you are aiming to develop a simple spatial Agent Based Model (ABM). This type of model is useful for studying behaviour and emergence in complex systems.

The ABM will represent individual 'agents' which will have varying characteristics. The model will run iteratively such that each agent will do things in simulation steps (iterations of the model) which represent the passage of time. What the agents do will change their characteristics, possibly the characteristics of other agents, and possibly the attributes of the spatial environment in which the agents can move.

In building an ABM, you will put theory into practise and learn about:

  • testing and debugging;
  • using 'for loops' and conditional 'if statements';/li>
  • defining functions and classes;
  • reading and writing data;
  • organising code into modules;
  • developing a simple GUI; and,
  • scraping and processing data from the Web.

The agents will have simple behaviour and the model will be very abstract. Nevertheless, hopefully, it will be clear how you can develop it into something more realistic and useful.

The 'environment' will be initialised with data read from a file. The 'model' simulation will be visualised as an animation. Additional outputs from the model will be saved in files.

2. Getting started

Initially two agents are going to be represented in two dimensional spatial locations given by cartesian coordinates: (x0, y0) and (x1, y1). 3. Randomly changing values

The Python standard library random module implements pseudo-random number generators for various distributions and provides a way to change variable values randomly. To use the module it has to be imported. Towards the top of 'model.py' add the following import statement:

import random

It is good practice to organise import statements at the top of files as this makes them easy to find and reduces inefficiencies of importing things multiple times.

An if statement is going to be used to increase or decrease 'x0' based on the value obtained from a call to the function random. First obtain and print a pseudo-random number in the range [0, 1) as follows:

rn = random.random()
print(rn)

Run your program a few times to observe that the value of 'rn' varies randomly.

What is happening, is that the start (seed) of pseudo-random sequence used to generate numbers randomly is being set from the computer clock time (which moves on incrementally at a high frequency.

If the value 'rn' is less that '0.5' increase 'x0' by '1', otherwise decrease 'x0' by '1':

if rn < 0.5:
    x0 = x0 + 1
else:
    x0 = x0 - 1
print("x0", x0)

Run your program a few times to observe that sometimes the value of 'x0' increases and sometimes it decreases and this corresponds to the value of 'rn'.

When testing code, it is usually undesireable to have different results each time. So, it is better to 'set the seed' of random. After the import statements add the following:

# Set the pseudo-random seed for reproducibility
random.seed(0)

When you run the program, you should get the following output: 

x0 0
y0 0
rn 0.8444218515250481
b False
x0 - 1

For different results pass in a different value into the 'seed' function (e.g. random.seed(1)).

Similarly modify the value of 'y0'. The similarly modify 'x1' and 'y1'.
